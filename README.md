## Console Based Banking System

This is a Simple console based Banking System using C++ which demonstrates the Basic Banking Features, Withdrawal, Transaction and Admin Functionality using the File Handling

## Features 
- Create New Account
- Login Access
- Transaction
- Admin Access
- View Account Details


## Files

- Bank.txt - Used to store User Details
- Admin.txt - Used to store Admin Access
- Detail.txt - plays a vital role in the process of Transaction
